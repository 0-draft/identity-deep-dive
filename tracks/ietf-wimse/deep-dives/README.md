# Deep Dives

Investigation notes for individual IETF WIMSE drafts and topics.

Scaffold a new topic from the shared template:

```bash
cp -R ../../../templates/deep-dive <topic-name>
```

Recommended workflow:

- `context.md`: What happened and why we are tracking it
- `spec-notes.md`: Spec differences and discussion points
- `interop-checklist.md`: Implementation verification perspectives
- `hands-on-log.md`: Experiment log
